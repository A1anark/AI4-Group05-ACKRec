# ACKRec - Attentional Graph Convolutional Networks for Knowledge Concept Recommendation

基于注意力图卷积网络的MOOCs知识概念推荐系统。

## 项目结构
AI4Edu-Group-ACKRec/
├── data/ # 存放数据样例
├── models/ # 模型定义代码
├── utils/ # 工具函数
├── scripts/ # 训练和数据处理脚本
├── saved_models/ # 训练好的模型权重
├── app.py # Streamlit Web界面
├── requirements.txt # 依赖库
├── config.py # 配置文件
├── quick_start.py # 快速启动测试
├── .gitignore # Git忽略文件
└── README.md # 项目说明
